# CONFIG-ONBOARDING

onboarding reference for `actual.cfg` binds and movement

---

## Movement

| Key | Action |
|-----|--------|
| W / A / S / D | Forward / Left / Back / Right |
| SPACE | Jump |
| mwheeldown | Jump (bhop scroll) |
| CAPSLOCK | Duck |
| CTRL | Duck |
| SHIFT | Walk (silent) |

---

## Grenades

| Key | Grenade |
|-----|---------|
| C | Smoke (slot6) |
| F | Flashbang (slot7) |
| G | HE grenade (slot8) |
| V | Decoy (slot10) |
| 4 | Cycle grenades |

**Jump throw:** jump then throw to extend smoke/flash range consistently (no longer a bind, but much easier to do)

---

## Utility

| Key | Action |
|-----|--------|
| Q | Last weapon (quick switch) |
| R | Reload |
| E | Use / interact |
| H | Drop weapon |
| MOUSE4 | Player ping |

---

## Practice Mode (nades.cfg)

| Key | Action |
|-----|--------|
| N | Noclip |
| L | Rethrow last grenade |
| I | Give full nade loadout + AK |
| O | Clear all nade projectiles + stop sound |

> Load with `exec nades` in a local server. Includes infinite ammo, respawn, $99999, 60min rounds.
